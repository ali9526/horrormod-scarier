# HorrorMod 2.1.0 — усиленная версия

Forge 1.20.1 хоррор-мод: случайные жуткие события, сущности-сталкеры
(Zero, Glitch, Formadia, ProInPvp), фейковые таблички, "распад" мира.

Исходники получены декомпиляцией оригинального .jar (исходного кода не было),
после чего вручную усилена частота событий и характеристики мобов.

## Что изменено по сравнению с оригиналом (2.0.0)

`com/horrormod/events/HorrorEvents.java`:
- интервалы всех случайных событий сокращены в 2–3 раза;
- глобальный кулдаун между событиями: 60 → 20 тиков;
- события начинаются почти сразу после захода игрока на сервер;
- "распад мира" стартует с 1-го дня вместо 5-го, происходит втрое чаще,
  затрагивает 6 блоков за раз вместо 2;
- падающий бедрок бьёт ближе к игроку и выше;
- красных факелов спавнится 5–10 вместо 2–5.

Сущности (`com/horrormod/entity/*.java`):
- здоровье, скорость и урон увеличены у Zero, Glitch, Formadia, ProInPvp;
- время "слежки" до атаки сильно сокращено;
- дальность обнаружения и телепортов увеличена;
- у Zero ослаблен порог зрительного контакта — поймать его взгляд легче.

## Сборка

Сборка идёт автоматически через GitHub Actions (`.github/workflows/build.yml`)
при пуше в `main` — готовый `.jar` появляется в Artifacts соответствующего
workflow run.

Локально: `./gradlew build` (нужен JDK 17), готовый jar — в `build/libs/`.
